Installation¶

BiaPy can be locally installed/run on any Linux, Windows and Mac OS platform using Docker or via command line (with Anaconda/Miniconda and Git). Alternatively, the BiaPy can also be used on Google Colab.

Command line installation¶

For command line usage, you only need to set-up a conda environment. For that you first need to install Anaconda/Miniconda. For a more detailed installation based on your operating system: Windows, macOS and Linux.

Afterwards, you will need git which is a free and open source distributed version control system. It will allow you to download the code easily with just a command. You can download and install it here. For a more detailed installation based on your operating system: Windows, macOS and Linux.

Once Anaconda and git are installed you need to use a terminal for the following steps. You can open a terminal in each of operating systems following these steps:

  • In Windows you have installed a terminal called Git Bash if you followed git installation above. Find that application in the Start menu by clicking on the Windows icon and typing Git Bash into the search bar. Click on it to open the Git Bash terminal.

  • In macOS you already have Bash terminal installed so just open it. If you have never done it before find more info here.

  • In Linux you already have Bash terminal installed so just open it. If you have never done it before find more info here.

Then you are prepared to download BiaPy repository by running this command in the terminal

git clone https://github.com/danifranco/BiaPy.git

This will create a folder called BiaPy that contains all the files of the library’s official repository. Then you need to create a conda environment using the file located in BiaPy/utils/env/environment.yml

conda env create -f BiaPy/utils/env/environment.yml
